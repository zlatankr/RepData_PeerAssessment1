Imputing missing values

  1. Calculate and report the total number of missing values in the dataset.
sum(is.na(File$steps))

## [1] 2304
  1. Devise strategy for imputing missing values. Impute NAs based on average number of steps take in that interval, rounding up.
merged <- merge(File,avgsteps,by.x="interval",by.y="interval",all=TRUE)

merged$cleansteps <- round(ifelse(is.na(merged$steps.x), 
                                  merged$cleansteps <- merged$steps.y,merged$cleansteps <- merged$steps.x))
  1. Create a new dataset ('merged') that is equal to the original dataset but with the missing data filled in.
